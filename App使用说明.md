### 肺功能仪

####Air Smart Extra系列

- VC模式下吹气
- MVV模式吹气
- FVC模式吹气

![WechatIMG55](https://tva1.sinaimg.cn/large/e6c9d24egy1h5z68zuv7lj208w0jrjs6.jpg)

#### AirSmart TT、Air Smart T1

> 提供三种吹气模式

|   MVV    |    VC    |   FVV    |
| :------: | :------: | :------: |
| PEF(L/s) | PEF(L/s) |  FVC(L)  |
| MVV(L/m) |  VC(L)   | FEV1(L)  |
|          |          | PEF(L/s) |
|          |          | FEV1/FVC |

![WechatIMG53](https://tva1.sinaimg.cn/large/e6c9d24egy1h5z6nl87hxj208w0jrmxq.jpg)

- Air Smart T1设备需要设置用户信息
  - 首次使用会有默认参数，用户可以自己修改
  - 选择任一模式进行吹气

### 呼吸训练仪

- 电量查询


- 开始训练
- 训练历史数据查询
  - 同步设备训练数据
- 呼吸训练仪设备系统设置
  - 设置各种参数
  - 清除历史数据
  - 重置设备系统

![WechatIMG51](https://tva1.sinaimg.cn/large/e6c9d24egy1h5z6bq54oyj209q0lmmxg.jpg)

  - 直喷
- 查询历史
  - 雾化历史数据记录和查询
