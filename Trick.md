VC模式下吹气

- **肺功能不全分级**

|      |   VC(%)   |
| :--: | :-------: |
| 基本正常 |    >80    |
| 轻度减退 | >70 &<=80 |
| 显著减退 | >50 &<=70 |
| 严重减退 | >20 &<=50 |
| 呼吸衰竭 |   <=20    |

- **肺容积异常的程度评价**

| 异常程度 |  VC(%)   |
| :--: | :------: |
|  正常  |   >=90   |
|  轻度  | >90&>=70 |
|  中度  | >70&>=50 |
|  重度  |   <50    |
