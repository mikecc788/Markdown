## GIT 基本理论

> 工作区域

git本地三个工作区域  工作目录（本地文件）、暂存区（add .）、资源库(本地仓库 commit) 、远程目录（push）

> 工作流程



## GIT文件操作

> 文件的四种状态

- Untracked：未跟踪
- Unmodify： 已经入库，未修改
- Modified ：已修改
- Staged 暂存状态

> 忽略文件

gitignore  空行或者以#开始的行将会被忽略

*.txt #忽略所有 .txt结尾的文件

## GIT分支

git分支中常用指令

```swift
git branch -r //列出所有远程分支
git branch [branch-name] //新建一个分支
git checkout -b [branch]// 切换分支
git merge [branch] 
```
